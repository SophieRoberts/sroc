The Course/Finish

How do I get from the start to the first control?

Well that's all part of the fun and just about the only thing we can't help you with!

Tip. Don't follow anyone else. The chances are they're either on a different course, or worse still, they could be lost.

Can I visit the controls in any order?

No, you must visit the controls in numerical order. If you don't you risk being disqualified.

What happens when I reach the first (second . . . ) control?

When you reach a control, check that the code on the control is correct by checking it against your control description sheet. If it's correct, punch the control.

Tip. Punching the wrong control is a very common cause of error, so checking the code could save you from being disqualified!

Tip. If you're using an e-card, ensure that the base unit beeps. Occasionally the base units fail. If you insert your e-card and it doesn't beep then it's likely that this has happened. Simply report this at the finish. There's a good chance that the organisers will know, as it's common for several courses to use the same controls.

What happens when I reach the finish?

If the event's using e-cards then there will be a finish unit. Punching this records your elapsed time for the course. Otherwise, you will be timed as you cross the finish line.

What happens if I retire?

Well we hope this doesn't happen, but in the unlikely event of it happening you must report to the finish. If you don't then the organisers could still be looking for you long after you've headed home!
