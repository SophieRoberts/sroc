Setting Out

The following five pages provide a step by step guide to competing in your first event. At the bottom of each page you'll find links to the 'next' and 'previous' pages.

When you reach the last page there is a Microsoft Word file that you can download. This file contains all the pages that appear on the website. This can be printed and taken to your first event to help you. Good luck and happy orienteering.

What's the best event to start with?

If there are no local events listed in the current Fixtures, you could try a Permanent Course. However, these guidelines explain all you need to know for a Colour Coded event. Check out the Event Types page if you're not sure what this means.

The fixture list will provide you with the location, i.e. grid reference, and OS map you'll need to navigate there.

What facilities are normally provided?

There will always be toilet facilities and generally one or more traders, selling O-kit and food. The bigger the event the more extensive the facilities.

What time should I arrive?

Registration is normally from 10:00 - 12:00, with start times from 10:30 - 12:30. However you should check this in the details for the event which you are interested in. Look in the fixtures list.
